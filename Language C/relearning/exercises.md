# Nivel 1: Sintaxis y Control de Flujo

1. Calculadora de 4 operaciones básicas - **Done**
2. Conversor de temperatura (Celsius ↔ Fahrenheit ↔ Kelvin)
3. Verificador de números primos
4. Generador de la serie Fibonacci (iterativo)
5. Palíndromo de strings (sin `string.h`)

# Nivel 2: Arrays y Strings

6. Implementa tu propia `strlen`, `strcpy`, `strcmp`
7. Invertir un array in-place
8. Ordenamiento: Bubble Sort, luego Selection Sort
9. Buscar elemento en array (lineal, luego binaria)
10. Rotación de array (izquierda/derecha por k posiciones)

# Nivel 3: Punteros

11. Swap de dos variables usando punteros
12. Suma de dos números usando solo punteros (sin variables directas)
13. Implementa `strcat` con aritmética de punteros
14. Recorre un array 2D usando solo punteros (sin `arr[i][j]`)
15. Implementa `strchr` y `strstr`

# Nivel 4: Memoria Dinámica

16. Array dinámico que crece automáticamente (tipo `vector` de C++)
17. Lista enlazada simple (insert, delete, search, print)
18. Lista doblemente enlazada
19. Pila (stack) usando lista enlazada
20. Cola (queue) usando lista enlazada

# Nivel 5: Estructuras de Datos Clásicas

21. Hash table simple (chaining para colisiones)
22. Árbol binario de búsqueda (insert, search, delete)
23. Recorridos de árbol (inorder, preorder, postorder)
24. Heap (min-heap o max-heap)
25. Merge sort y quicksort

# Nivel 6: Manejo de Archivos

26. Leer archivo de texto línea por línea y contar palabras
27. Copiar un archivo binario byte por byte
28. Parser de CSV simple
29. `grep` básico (buscar patrón en archivo)
30. “Base de datos” en archivo (insertar/buscar registros)

# Nivel 7: Bits y Bajo Nivel

31. Contar bits activados (popcount)
32. Verificar potencia de 2 (con operaciones de bits)
33. Invertir bits de un número
34. XOR cipher simple
35. Compresión/descompresión RLE (Run-Length Encoding)

# Nivel 8: Proyectos Integradores

36. Mini-shell: ejecuta comandos básicos y maneja pipes
37. Allocator de memoria: tu propio `malloc`/`free` básico
38. Servidor TCP echo (acepta conexiones y devuelve lo recibido)
39. Intérprete de Brainfuck
40. Compresor LZ77 básico

## Reglas del juego

1. Empieza en el nivel que te quede cómodo.
2. No copies código de internet (solo documentación).
3. Compila con warnings: `-Wall -Wextra -Werror`.
4. Usa Valgrind desde el nivel 4 en adelante.
5. Escribe tests para cada función antes de integrar.
